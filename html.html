Exemplos de Código

Aqui estão alguns exemplos de código HTML para você experimentar:

        <!DOCTYPE html>
        <html>
        <head>
            <title>Minha Página HTML</title>
        </head>
        <body>
            <h1>Título</h1>
            <p>Este é um parágrafo.</p>
        </body>
        </html>
